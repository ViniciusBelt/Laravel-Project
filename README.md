## Instalação

docker-compose up --build

<br>
![db](https://github.com/ViniciusBelt/imgs/blob/master/db.png?raw=true)

php artisan serve

php artisan migrate:refresh --seed

## Requisitos

- PHP (Descomentar pdo_mysql).
- Docker.
- Composer.
- Ferramenta para DB (DBeaver).
